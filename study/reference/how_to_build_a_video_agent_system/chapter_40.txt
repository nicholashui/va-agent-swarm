### High-Level Summary of the Review Aggregator Agent

This is a powerful AI system designed to automatically monitor, collect, and analyze movie reviews from multiple online platforms. Think of it as a "review intelligence network" that transforms scattered opinions from hundreds of reviewers into organized, actionable insights for movie studios, distributors, and marketing teams.

### Key Concepts Explained Simply
- **Sentiment Analysis**: Computer programs read reviews to determine if they're positive, negative, or neutral—like teaching a computer to understand if reviewers love or hate the movie.
- **Topic Modeling**: A technique that automatically identifies common themes in reviews, such as "stunning visuals" or "weak plot," without manual coding. It's like the system reading thousands of reviews and naturally grouping similar opinions.
- **Web Scraping**: Automated programs that extract data from review sites—like copy-pasting, but at massive scale.
- **BERT (Bidirectional Encoder Representations from Transformers)**: An advanced AI model that understands language context. Unlike older systems that read word-by-word, BERT reads sentences bidirectionally, grasping meaning like humans do.

### Main Features
1. **Multi-Platform Review Collection**: Automatically gathers reviews from Rotten Tomatoes, Metacritic, IMDb, Letterboxd, and major newspapers using advanced scraping techniques that evade detection while respecting site rules. Handles over 100 sources continuously.
2. **Advanced Sentiment Analysis**: Goes beyond overall sentiment to analyze specific aspects like acting, visuals, plot, and music. Detects emotions, sarcasm, and mixed feelings, with confidence scores for each analysis.
3. **Intelligent Topic Discovery**: Automatically uncovers emerging themes and trends, groups similar opinions without manual classification, and tracks how consensus evolves over time.
4. **Critic Profiling System**: Builds behavioral profiles for each critic, tracking their scoring patterns, genre preferences, and biases—prioritizing high-influence critics.
5. **Real-Time Monitoring**: Alerts when scores cross key thresholds (e.g., 60%, 75%, 90%), notifies on new reviews from major critics, and detects viral reviews blowing up on social media.

### Key Highlights
- **Comprehensive Coverage**: Monitors 100+ sources simultaneously, from major outlets to independent blogs, ensuring no important voices are missed.
- **Multi-Layer Analysis**: Beyond simple positive/negative, dives into plot, acting, visuals, music, directing—with accuracy over 92%.
- **Weighted Intelligence**: Not all reviews are equal—top outlets like NYT and Variety, plus influential critics, get higher weight for more accurate aggregate scores.
- **Real-Time Insights**: Instant alerts for threshold breaches or big critic reviews, enabling quick marketing responses.
- **Predictive Power**: Uses machine learning to forecast box office potential, awards viability, and long-term cultural impact.
- **Aspect-Based Insights**: Pinpoints what critics praise or criticize, e.g., "weak plot but explosive visuals," for targeted marketing.
- **Trend Detection**: Automatically spots new themes and consensus shifts without manual coding.

### Numbers and Data
- **Processing Power**: Continuously monitors 100+ sources; sentiment classification accuracy >92%; processes thousands of reviews in minutes.
- **Analysis Depth**: Covers 7 movie aspects (plot, acting, visuals, directing, music, technical, entertainment); 5-tier outlet weighting (top-tier 5x, general audience 1x); sentiment scores from -1 to +1 with confidence ratings.
- **Technical Specs**: BERT model with >110M parameters; analyzes up to 512 tokens per review; requires at least 15 reviews to form topic clusters.
- **Performance Metrics**: Ensemble methods boost negative sentiment detection accuracy by 3.6%; full scans in 60 seconds; complete reports in 3-5 minutes.

### Who It's For and Why They Care
- **Studios & Distributors**: Track release-period reviews in real-time, extract marketable quotes and positive themes, spot issues early to protect box office, predict awards momentum.
- **Marketing Teams**: Pull strong quotes for promo materials, understand which aspects resonate, monitor competitors, react to viral reviews or shifts.

# Review Aggregator Agent - The Critic Whisperer

## What to Build

The Review Aggregator Agent is a sophisticated multi-platform intelligence system designed to monitor, analyze, and interpret critical reception of films across the digital landscape. The system comprises several core components.

<#0.5#>

### Advanced Review Scraper Architecture

The system uses **Scrapy**, an open-source web crawling framework, to build a robust scraping infrastructure for large-scale data extraction. It targets major review platforms, including:

- **Rotten Tomatoes**: Captures Tomatometer scores, critic reviews, audience ratings, and freshness classifications using automated document organization and topic extraction.
- **Metacritic**: Extracts Metascores, weighted critic reviews, user reviews, and comprehensive metadata.
- **IMDb**: Harvests user ratings, professional critic scores, and detailed review text from the world's largest movie database.
- **Letterboxd**: Collects star ratings, detailed reviews, and social engagement metrics from the cinephile community.
- **Entertainment Weekly**: Gathers professional critic reviews and industry insights.
- **The Hollywood Reporter**: Retrieves trade publication reviews with industry perspectives.
- **Variety**: Extracts professional entertainment industry reviews and analysis.

Scrapy's asynchronous requests, parallel crawling, and built-in data handling enable efficient processing of millions of records. Its architecture supports highly concurrent logic with callbacks for task completion.

<#0.5#>

### Intelligent Sentiment Analysis Engine

The sentiment analysis engine performs opinion mining to analyze emotions and opinions in text, supporting social media monitoring, product recognition, customer satisfaction, and acceptance evaluation. It employs multiple approaches:

- **BERT-Based Analysis**: Fine-tunes **Bidirectional Encoder Representations from Transformers (BERT)** with **Bidirectional Long Short-Term Memory (BiLSTM)** for movie review sentiment analysis, achieving over 92% accuracy in classifying reviews as positive or negative—surpassing state-of-the-art methods.
- **Aspect-Based Sentiment Analysis**: Compares **Term Frequency-Inverse Document Frequency (TF-IDF)** and BERT for sentiment analysis, integrating traditional and transformer-based methods. A multi-self-attention BERT model identifies contextual relations across word sequences, incorporating negations and intensifications for precise polarity prediction.

<#0.5#>

### Advanced Quote Extraction System

The quote extraction component uses **Natural Language Processing (NLP)** to identify marketable soundbites from reviews. It assigns polarity ratings from -1 (negative) to 1 (positive) and subjectivity scores from 0 (factual) to 1 (subjective). BERT-CNN approaches improve sentiment detection, particularly for negative sentiments, with 6% accuracy gains via ensemble methods.

<#0.5#>

### Comprehensive Critic Profiling System

The critic profiler builds detailed behavioral models of reviewers, tracking:

- **Historical Review Patterns**: Average scores, score distributions, and consistency metrics.
- **Genre Preferences**: Scoring tendencies across movie genres.
- **Influence Scoring**: Weighted impact based on publication prestige and reach.
- **Bias Detection**: Identification of systematic preferences or prejudices.
- **Temporal Analysis**: Changes in reviewing patterns over time.

<#0.5#>

### Topic Modeling and Trend Detection

The system uses BERT and **class-based TF-IDF (c-TF-IDF)** to generate dense, interpretable topic clusters while preserving key words in descriptions. **BERTopic** integrates six customizable modules for coherent results.

- **BERTopic Integration**: Combines BERT-based language models with **Uniform Manifold Approximation and Projection (UMAP)** and **Hierarchical Density-Based Spatial Clustering of Applications with Noise (HDBSCAN)** for advanced topic modeling. The process includes document embedding with pre-trained transformers, dimensionality reduction, clustering of similar documents, and topic extraction using c-TF-IDF.
- **Dynamic Topic Discovery**: Merges similar topics based on cosine similarity between c-TF-IDF vectors, iteratively reducing topics if similarity exceeds a threshold of 0.9 or automatically determining optimal topic counts.

<#0.5#>

### Real-Time Impact Scoring

The impact scorer weights critical opinions by:

- **Publication Tier**: Top-tier outlets (e.g., *The New York Times*, *Variety*) receive 5× weight.
- **Critic Reputation**: Scores based on career achievements.
- **Social Media Reach**: Twitter followers, engagement rates, and viral potential.
- **Historical Accuracy**: Past prediction accuracy for box office and awards success.
- **Audience Alignment**: Correlation between critic and audience scores.

<#0.5#>

### Intelligent Alert System

The alert system delivers real-time notifications for:

- **Score Threshold Changes**: Rotten Tomatoes Tomatometer crossing 60%, 75%, or 90%.
- **Major Critic Reviews**: From prominent critics (e.g., Peter Travers, A.O. Scott).
- **Viral Review Detection**: Reviews gaining significant social media traction.
- **Competitive Analysis**: Comparisons with similar films or franchise entries.
- **Awards Season Momentum**: Shifts in critical consensus during awards campaigns.

## What It Does

<#0.5#>

### Comprehensive Multi-Platform Scraping

The system gathers training and research data via web scraping to meet high-volume sentiment analysis needs. Similar to tools in finance that extract insights from news articles for investment strategies, it operates as a distributed intelligence network monitoring over 100 review sources. It employs rate limiting, anti-detection measures, proxy rotation, and custom HTTP headers to evade bot detection.

- **Automated Source Discovery**: Dynamically identifies new sources through RSS feed monitoring, social media analysis, blog network scanning, and academic database searches.
- **Intelligent Scheduling**: Establishes scraping schedules based on film release calendars, embargo dates, historical posting patterns, breaking news, and awards season intensity. Embargo-aware scheduling captures reviews immediately upon public availability.

<#0.5#>

### Advanced Sentiment Analysis Processing

The engine analyzes opinions for news, products, and campaigns, aiding brand analysis, customer satisfaction, and acceptance evaluation.

- **Multi-Model Ensemble Approach**: Integrates **Transformers** (from Hugging Face) and **TextBlob**, with Transformers offering superior performance due to active maintenance. **DistilBERT** classifies text as positive or negative, providing confidence probabilities.
- **Contextual Understanding**: BERT embeddings capture semantic meaning for accurate topic modeling and interpretability via keyword and document assignments. It processes sarcasm, comparative analysis, cultural references, emotional intensity, and mixed sentiments.
- **Aspect-Level Analysis**: Involves preprocessing, **Word2Vec** feature extraction, and sentiment classification. It evaluates:
  - **Plot/Story**: Narrative structure, pacing, character development.
  - **Acting/Performance**: Individual and ensemble quality.
  - **Visual Elements**: Cinematography, special effects, production design.
  - **Audio**: Musical score, sound design, dialogue quality.
  - **Direction**: Filmmaking vision and execution.
  - **Technical Craft**: Editing, lighting, costume design.

<#0.5#>

### Intelligent Quote Extraction and Curation

The system uses NLP to identify impactful, marketable statements. Topic modeling shows negative reviews often critique multiple aspects, complicating sentiment analysis.

- **Marketability Scoring Algorithm**: Scores quotes based on superlative density, emotional impact, quotability, context independence, and source credibility.
- **Automated Categorization**: Sorts quotes into:
  - **Pull Quotes**: Short, punchy statements for posters.
  - **Comparison Quotes**: References to other films.
  - **Award Predictions**: Oscar or award potential.
  - **Audience Recommendations**: Appeals to specific segments.
  - **Visual Praise**: Commendations of cinematography or effects.

<#0.5#>

### Sophisticated Critic Profiling and Analysis

The profiler uses deep learning and transformers (BERT, GPT) to detect positive, negative, or neutral sentiment, handling context, sarcasm, and mixed sentiments.

- **Behavioral Pattern Recognition**: Builds profiles covering scoring tendencies, genre preferences, career trajectory, influence networks, and prediction accuracy.
- **Bias Detection Algorithms**: Identifies studio relationships, genre prejudices, star preferences, cultural blind spots, and temporal shifts via statistical analysis.

<#0.5#>

### Real-Time Trend Detection and Analysis

The system applies unsupervised learning for document tagging, survey analysis, and content organization.

- **Dynamic Topic Modeling**: Uses modified c-TF-IDF to identify relevant terms within clusters. It tracks critical consensus, audience-critic divergence, genre evolution, cultural moments, and technical innovations.
- **Predictive Analytics**: Models predict box office correlation, awards momentum, cultural longevity, franchise implications, and streaming performance.

<#0.5#>

### Comprehensive Impact Scoring and Weighting

The scorer calculates nuanced influence in the film criticism ecosystem.

- **Multi-Tier Influence Architecture**:
  - **Tier 1 (5× Weight)**: Major newspapers and trade publications.
  - **Tier 2 (3× Weight)**: Regional and specialized entertainment outlets.
  - **Tier 3 (2× Weight)**: Established online publications and respected critics.
  - **Tier 4 (1× Weight)**: Blogs, emerging voices, and audience reviews.
- **Dynamic Weighting Adjustments**: Recalibrates based on viral impact, citation frequency, audience engagement, industry recognition, and prediction accuracy.

### Intelligent Alert and Notification System

The system automates workflows by saving data to Google Drive, sending Gmail and Slack notifications, and monitoring scheduled runs.

- **Multi-Channel Alert Distribution**:
  - **Real-Time Dashboards**: Live interfaces for monitoring teams.
  - **Email Digests**: Customizable summaries for stakeholders.
  - **SMS Alerts**: Urgent threshold notifications.
  - **Slack Integration**: Team notifications with analysis.
  - **API Webhooks**: Integration with external systems.
- **Intelligent Filtering and Prioritization**:
  - **Significance Scoring**: Automated importance assessment.
  - **Stakeholder Targeting**: Customized based on interests.

  - **Context Enhancement**: Notifications with trends.

### Core Implementation Architecture

#### 1. Web Scraping Infrastructure - Scrapy Framework

**Approach**: Scrapy enables asynchronous crawling with Twisted for concurrency. It includes data validation, storage, and HTML parsing.

**Code Architecture**:
```python
# Multi-platform scraping architecture
class ReviewAggregatorPipeline:
    def __init__(self):
        self.scrapers = {
            'rotten_tomatoes': RottenTomatoesSpider,
            'metacritic': MetacriticSpider,
            'imdb': IMDbSpider
        }
```

**Pros**:
- Asynchronous engine for scale.
- Built-in retry, validation, and output formats.
- Extensions for proxies and emulation.

**Cons**:
- Requires anti-detection for blocking.
- Needs extensions for JavaScript.
- Asynchronous patterns add complexity.

#### 2. Sentiment Analysis Engine - BERT/Transformers

**Approach**: BERT provides bidirectional context for classification tasks.

**Code Architecture**:
```python
from transformers import BertForSequenceClassification, BertTokenizer

class ReviewSentimentAnalyzer:
    def __init__(self):
        self.tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
        self.model = BertForSequenceClassification.from_pretrained(
            'bert-base-uncased', 
            num_labels=3  # positive, negative, neutral
        )
```

**Pros**:
- Strong transfer learning.
- High accuracy with context.
- Pre-trained models via Hugging Face.

**Cons**:
- High computational cost.
- 110M+ parameters limit interpretability.
- Requires tokenization and masking.

#### 3. Topic Modeling System - BERTopic

**Approach**: BERTopic uses transformers and c-TF-IDF for coherent topics via sentence-transformers, UMAP, HDBSCAN, and c-TF-IDF.

**Code Architecture**:
```python
from bertopic import BERTopic
from sentence_transformers import SentenceTransformer
from umap import UMAP
from hdbscan import HDBSCAN

class ReviewTopicAnalyzer:
    def __init__(self):
        self.embedding_model = SentenceTransformer('all-MiniLM-L6-v2')
        self.topic_model = BERTopic(
            embedding_model=self.embedding_model,
            min_topic_size=15
        )
```

**Pros**:
- Modular components.
- Semantic coherence.
- Visualization support.

**Cons**:
- Guided modeling needs accurate seeds.
- Subjective evaluation.
- Clustering requires tuning.

#### 4. Data Storage Architecture - MongoDB + Elasticsearch

**Code Architecture**:

class HybridDataStore:
