# Makeup & Hair Agent - The Face Sculptor

## Overview

This document outlines the design, functionality, and implementation of the **Makeup & Hair Agent**, a comprehensive system for advanced character makeup design, hair simulation, aging progression, injury simulation, and cultural authenticity validation. It serves as a digital makeup artist and stylist for virtual production pipelines, ensuring realistic, consistent, and historically accurate results across scenes.

<#0.5#>

## What to Build

The system comprises the following core components:

- **Advanced Character Makeup Designer**: Handles natural beauty makeup, theatrical transformations, special effects, age progression, injury simulation, and cultural/period authenticity.
- **Procedural Skin Shader System**: Implements multi-layered subsurface scattering (SSS) using physically based models, separating surface reflection from subsurface light transport to achieve soft, realistic skin appearance via light penetration and dermal scattering.
- **Hair Physics Simulation Platform**: Utilizes an advanced Vellum solver for physics-based simulations of hair movement, bending, and object interactions, enforcing distance and orientation constraints with adjustable bend stiffness and damping ratios.
- **AI-Powered Aging Simulator**: Employs a multi-network architecture (encoder, FaceNet, generator, discriminator) with ethnicity-specific aging data and weak spatial supervision for realistic progression and regression.
- **Procedural Injury/Wound Generator**: Simulates wound progression with healing timelines, bruise color evolution, scar formation, and blood flow dynamics.
- **Comprehensive Continuity Tracker**: Manages frame-by-frame makeup and hair states to ensure consistency across scenes and temporal changes.
- **Cultural/Period Authenticity Validator**: Verifies historical makeup and hairstyles against extensive reference databases.

## What It Does

<#0.5#>

### Core Makeup Design System

The agent functions as a digital makeup artist, applying virtual cosmetics with high realism. It generates skin pores using procedural noise (e.g., black-and-white spots and clouds) with adjustable scale, adhering to physically based rendering (PBR) principles: non-metallic materials (metallic value = 0), emphasizing base color, roughness, normal/height mapping, and SSS balance.

#### Advanced Skin Texturing Workflow

The system uses Smart Materials and procedural effects in tools like Substance Painter, layering artistically and refining iteratively under varied lighting. It incorporates pre-built materials (e.g., "Skin Human") and community assets (e.g., "Realistic Human Skin") with tri-planar projection for adjustable base pores and maskable layers for color/detail control.

Textures separate base color from roughness and scattering data for non-destructive editing, allowing isolated adjustments (e.g., skin tone) without full retexturing.

#### Procedural Pore Generation

Pores are generated with controlled size, depth, and variation, referencing high-resolution images and using ZBrush-style layering for depth. Skin direction includes micro-creases from facial movement; layer intensity simulates age (deeper creases for older characters).

<#0.5#>

### Subsurface Scattering Implementation

#### Physical Accuracy

The skin shader models SSS where light penetrates the surface, scatters and absorbs subsurface, then exits, producing translucency for soft skin. About 6% of light reflects directly via Fresnel at the oily surface; the rest undergoes subsurface scattering and absorption.

#### Multi-Layer Skin Model

It simulates up to five layers with varying absorption and scattering, supported by graphics research. The bidirectional scattering-surface reflectance distribution function (BSSRDF) softens appearance and enables color bleeding in shadows, with blood absorption reddening deep-scattered light.

#### Real-Time Performance

Full physical accuracy exceeds real-time limits, so the system approximates visuals using lighting blur (e.g., Gaussian blurs for "glow" effects) in post-processing.

<#0.5#>

### Hair Simulation System

#### Vellum Physics Engine

Hair uses position-based dynamics (PBD) via Houdini's Vellum framework, integrated with cloth, grains, fluids, and soft bodies. It maintains segment distance and orientation, with bend stiffness and damping preventing twisting; anti-twist support avoids axis rotation.

#### Advanced Constraint Systems

Constraints combine edge-distance and bend-angle calculations, distinguishing twist/stretch from regular geometry. Higher bend stiffness and lower damping yield stiffer hair; fewer segments and more iterations increase rigidity.

#### Collision and Interaction

Surface colliders handle deforming objects without VDB generation for efficiency. For long hair, disable self-collisions and use glue constraints for clumping, reducing chaos.

#### Pin and Root Systems

Roots pin to scalp via Group Expression SOPs (points, first primitive point). Orientation pinning preserves groomed direction; soft pins allow flex around roots.

<#0.5#>

### Aging Simulation Technology

#### GAN-Based Architecture

The system uses a generator (G) and discriminator (D) per deep convolutional GAN (DCGAN), with conditional inputs at G and D's first layer. Aging occurs via latent vector approximation and age switching.

#### Multi-Network Integration

Age-conditional GAN (age-cGAN) includes encoder, FaceNet, generator, and discriminator. Pretrained Inception-ResNet-v2 yields 128D embeddings via Euclidean distance on real/reconstructed images.

#### Aging Progression Control

It labels 15 clinical signs across 7,000 images for ethnicity-specific maps, controlling localization and degree. Outputs include wrinkles, hairline changes, and super-resolution while preserving identity.

#### Training Methodology

Optimized with ADAM over 100 epochs across six age groups (0-18, 19-29, 30-39, 40-49, 50-59, 60+). G minimizes generation-to-target loss; D maximizes real-vs-generated discrimination until outputs are indistinguishable.

<#0.5#>

### Virtual Makeup Technology

#### Face Tracking Precision

Detects up to 151 landmarks for precise placement, refining alignment on contours. Multi-point tracking handles movement for realistic try-ons in images/videos.

#### Real-Time Application

Tracks points through video, adjusting for rotations/translations to maintain positioning in dynamic interactions.

#### Makeup Realism

Aligns products (e.g., lipstick on lips, eyeshadow on creases) dynamically with expressions, ensuring seamless motion without artifacts.

### Advanced Face Tracking and Landmark Detection

#### Primary: OpenCV + Dlib

Dlib's pre-trained model detects 68 landmarks (e.g., eyes, lips, nose).

```python
import cv2
import dlib
import numpy as np

class FacialLandmarkTracker:
    def __init__(self):
        self.detector = dlib.get_frontal_face_detector()
        self.predictor = dlib.shape_predictor('shape_predictor_68_face_landmarks.dat')

    def detect_landmarks(self, image):
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
        faces = self.detector(gray)
        landmarks = []
        for face in faces:
            shape = self.predictor(gray, face)
            landmarks.append([(p.x, p.y) for p in shape.parts()])
        return landmarks
```

**Pros**: Real-time C++ performance; extensive support; free.  
**Cons**: Adult-focused; 68 landmarks only; ethnicity variance.  
**Integration**: OpenCV for capture; Unity via plugins.

### Physically-Based Skin Rendering with Subsurface Scattering

#### Primary: Custom Unity HDRP Shader

Uses texture-space SSS with UV Gaussian kernels.

```hlsl
Shader "Custom/SubsurfaceScatteringSkin"
{
    Properties
    {
        _BaseColor ("Base Color", Color) = (1,1,1,1)
        _SubsurfaceScatteringProfile ("SSS Profile", 2D) = "white" {}
        _ScatteringRadius ("Scattering Radius", Range(0, 10)) = 5
        _ThicknessTex ("Thickness", 2D) = "white" {}
    }
    SubShader { Tags { "RenderPipeline"="UniversalPipeline" } }
    Pass
    {
        Name "ForwardLit"
        HLSLPROGRAM
        #pragma vertex vert
        #pragma fragment frag
        #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
        #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"

        struct Attributes { float4 positionOS : POSITION; float3 normalOS : NORMAL; float2 uv : TEXCOORD0; };
        struct Varyings { float4 positionHCS : SV_POSITION; float3 positionWS : TEXCOORD0; float3 normalWS : TEXCOORD1; float2 uv : TEXCOORD2; };

        Varyings vert(Attributes input) {
            Varyings output;
            VertexPositionInputs pos = GetVertexPositionInputs(input.positionOS.xyz);
            output.positionHCS = pos.positionCS;
            output.positionWS = pos.positionWS;
            output.normalWS = GetWorldSpaceNormalizeMix(input.normalOS, 0.0);
            output.uv = input.uv;
            return output;
        }

        float3 SubsurfaceScattering(float3 lightDir, float3 normal, float3 viewDir, float thickness, float2 uv) {
            float transmission = pow(saturate(dot(-lightDir, normal + viewDir)), _ScatteringRadius) * thickness;
            float3 scatteringProfile = tex2D(_SubsurfaceScatteringProfile, float2(transmission, 0.5)).rgb;
            return scatteringProfile;
        }

        half4 frag(Varyings input) : SV_Target {
            float3 normalWS = normalize(input.normalWS);
            Light mainLight = GetMainLight();
            float3 viewDirWS = GetWorldSpaceViewDir(input.positionWS);
            float thickness = tex2D(_ThicknessTex, input.uv).r;
            float3 sss = SubsurfaceScattering(mainLight.direction, normalWS, viewDirWS, thickness, input.uv);
            half3 color = _BaseColor.rgb + sss * mainLight.color;
            return half4(color, 1.0);
        }
        ENDHLSL
    }
}
```

}

### Hair Physics Simulation System

#### Primary: Custom Vellum-Inspired in Unity

Uses PBD with Verlet integration and jobs.

```csharp
using Unity.Collections;
using Unity.Jobs;
using UnityEngine;

public class HairStrandSimulation : MonoBehaviour
{
    [SerializeField] private int segmentsPerStrand = 10;
    [SerializeField] private float strandLength = 1f;
    [SerializeField] private float damping = 0.15f;
    [SerializeField] private float stiffness = 100f;

    private NativeArray<Vector3> positions;
    private NativeArray<Vector3> oldPositions;
    private NativeArray<float> masses;

    void Start()
    {
        int totalPoints = segmentsPerStrand * /* num strands */;
        positions = new NativeArray<Vector3>(totalPoints, Allocator.Persistent);
        oldPositions = new NativeArray<Vector3>(totalPoints, Allocator.Persistent);
        masses = new NativeArray<float>(totalPoints, Allocator.Persistent);
        // Initialize positions, etc.
    }

    struct HairPhysicsJob : IJobParallelFor
    {
        [WriteOnly] public NativeArray<Vector3> positions;
        [ReadOnly] public NativeArray<Vector3> oldPositions;
        [ReadOnly] public NativeArray<float> masses;
        public float deltaTime;
        public float damping;
        public Vector3 gravity;

        public void Execute(int index)
        {
            if (index == 0) return; // Root pinned

            Vector3 velocity = (positions[index] - oldPositions[index]) / deltaTime;
            Vector3 force = gravity * masses[index];

            // Distance constraint
            Vector3 delta = positions[index] - positions[index - 1];
            float distance = delta.magnitude;
            float restLength = strandLength / (segmentsPerStrand - 1);
            if (distance > restLength)
            {
                Vector3 correction = delta.normalized * (distance - restLength) * 0.5f;
                positions[index] -= correction;
                positions[index - 1] += correction;
            }

            // Verlet
            Vector3 newPos = positions[index] + velocity * damping + force * deltaTime * deltaTime / masses[index];
            oldPositions[index] = positions[index];
            positions[index] = newPos;
        }
    }

    void Update()
    {
        var job = new HairPhysicsJob
        {
            positions = positions,
            oldPositions = oldPositions,
            masses = masses,
            deltaTime = Time.deltaTime,
            damping = damping,
            gravity = Physics.gravity
        };
        JobHandle handle = job.Schedule(positions.Length, 64);
        handle.Complete();
    }

    void OnDestroy()
    {
        positions.Dispose();
        oldPositions.Dispose();
        masses.Dispose();
    }
}
```

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch.utils.data import DataLoader

class Generator(nn.Module):
    def __init__(self, num_channels=64):
        super().__init__()
        self.encoder = nn.Sequential(
            nn.Conv2d(3, num_channels, 4, 2, 1),
            nn.LeakyReLU(0.2),
            nn.Conv2d(num_channels, num_channels*2, 4, 2, 1),
            nn.BatchNorm2d(num_channels*2),
            nn.LeakyReLU(0.2),
            # Add more layers up to 512 channels
        )
        self.decoder = nn.Sequential(
            nn.ConvTranspose2d(num_channels*8, num_channels*4, 4, 2, 1),
            nn.BatchNorm2d(num_channels*4),
            nn.ReLU(),
            # Add more layers to 3 channels
            nn.Tanh()
        )
        self.age_modulator = nn.Linear(6, num_channels*8)  # 6 age groups

    def forward(self, x, age_label):
        encoded = self.encoder(x)
        age_feat = F.relu(self.age_modulator(age_label))
        modulated = encoded + age_feat.view(encoded.shape)
        return self.decoder(modulated)

class Discriminator(nn.Module):
    def __init__(self):
        super().__init__()
        self.model = nn.Sequential(
            nn.Conv2d(3 + 1, 64, 4, 2, 1),  # +1 for age label
            nn.LeakyReLU(0.2),
            # More conv layers
            nn.Conv2d(512, 1, 4, 1, 0),
            nn.Sigmoid()
        )

    def forward(self, x, age_label):
        age_map = age_label.view(-1, 1, 1, 1).expand(-1, 1, x.shape[2], x.shape[3])
        input = torch.cat([x, age_map], dim=1)
        return self.model(input)

# Training
def train_age_gan(generator, discriminator, dataloader, epochs=100):
    optimizer_G = torch.optim.Adam(generator.parameters(), lr=0.0002, betas=(0.5, 0.999))
    optimizer_D = torch.optim.Adam(discriminator.parameters(), lr=0.0002, betas=(0.5, 0.999))
    criterion = nn.BCELoss()

    for epoch in range(epochs):
        for real_imgs, ages in dataloader:
            # Train D
            real_labels = torch.ones(real_imgs.size(0), 1)
            fake_imgs = generator(real_imgs, ages)
            fake_labels = torch.zeros(real_imgs.size(0), 1)

            d_real = discriminator(real_imgs, ages)
            loss_d_real = criterion(d_real, real_labels)
            d_fake = discriminator(fake_imgs.detach(), ages)
            loss_d_fake = criterion(d_fake, fake_labels)
            loss_d = (loss_d_real + loss_d_fake) / 2
            optimizer_D.zero_grad()
            loss_d.backward()
            optimizer_D.step()

            # Train G
            g_output = discriminator(fake_imgs, ages)
            loss_g = criterion(g_output, real_labels)
            optimizer_G.zero_grad()
            loss_g.backward()
            optimizer_G.step()
```

**Cons**: Compute-heavy training; input sensitivity.  
**Integration**: Python.NET to Unity; GPU inference.

            for idx in upper_lip + lower_lip:
                x = int(landmarks.landmark[idx].x * w)
                y = int(landmarks.landmark[idx].y * h)
                lip_points.append([x, y])

            cv2.fillPoly(mask, [np.array(lip_points)], 255)

            foreground[mask == 255] = color
            result = cv2.addWeighted(image, 0.7, foreground, 0.3, 0)
            return result

import cv2
import dlib
import numpy as np

class FacialLandmarkTracker:

{
    Properties

        _SubsurfaceScatteringProfile ("SSS Profile", 2D) = "white" {}
        _ScatteringRadius ("Scattering Radius", Range(0, 10)) = 5
        _ThicknessTex ("Thickness", 2D) = "white" {}

    Pass
    {

        #pragma vertex vert

        #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
        #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"

        struct Attributes { float4 positionOS : POSITION; float3 normalOS : NORMAL; float2 uv : TEXCOORD0; };
        struct Varyings { float4 positionHCS : SV_POSITION; float3 positionWS : TEXCOORD0; float3 normalWS : TEXCOORD1; float2 uv : TEXCOORD2; };

        Varyings vert(Attributes input) {
            Varyings output;
            VertexPositionInputs pos = GetVertexPositionInputs(input.positionOS.xyz);
            output.positionHCS = pos.positionCS;
            output.positionWS = pos.positionWS;
            output.normalWS = GetWorldSpaceNormalizeMix(input.normalOS, 0.0);
            output.uv = input.uv;
            return output;
        }

        float3 SubsurfaceScattering(float3 lightDir, float3 normal, float3 viewDir, float thickness, float2 uv) {
            float transmission = pow(saturate(dot(-lightDir, normal + viewDir)), _ScatteringRadius) * thickness;
            float3 scatteringProfile = tex2D(_SubsurfaceScatteringProfile, float2(transmission, 0.5)).rgb;
            return scatteringProfile;
        }

        half4 frag(Varyings input) : SV_Target {
            float3 normalWS = normalize(input.normalWS);
            Light mainLight = GetMainLight();
            float3 viewDirWS = GetWorldSpaceViewDir(input.positionWS);
            float thickness = tex2D(_ThicknessTex, input.uv).r;
            float3 sss = SubsurfaceScattering(mainLight.direction, normalWS, viewDirWS, thickness, input.uv);
            half3 color = _BaseColor.rgb + sss * mainLight.color;
            return half4(color, 1.0);
        }
        ENDHLSL
    }
}

```csharp
using Unity.Collections;
using Unity.Jobs;
using UnityEngine;

public class HairStrandSimulation : MonoBehaviour
{
    [SerializeField] private int segmentsPerStrand = 10;
    [SerializeField] private float strandLength = 1f;
    [SerializeField] private float damping = 0.15f;
    [SerializeField] private float stiffness = 100f;

    private NativeArray<Vector3> positions;
    private NativeArray<Vector3> oldPositions;
    private NativeArray<float> masses;

    void Start()
    {
        int totalPoints = segmentsPerStrand * /* num strands */;
        positions = new NativeArray<Vector3>(totalPoints, Allocator.Persistent);
        oldPositions = new NativeArray<Vector3>(totalPoints, Allocator.Persistent);
        masses = new NativeArray<float>(totalPoints, Allocator.Persistent);
        // Initialize positions, etc.
    }

    struct HairPhysicsJob : IJobParallelFor
    {
        [WriteOnly] public NativeArray<Vector3> positions;
        [ReadOnly] public NativeArray<Vector3> oldPositions;
        [ReadOnly] public NativeArray<float> masses;
        public float deltaTime;
        public float damping;
        public Vector3 gravity;

        public void Execute(int index)
        {
            if (index == 0) return; // Root pinned

            Vector3 velocity = (positions[index] - oldPositions[index]) / deltaTime;
            Vector3 force = gravity * masses[index];

            // Distance constraint
            Vector3 delta = positions[index] - positions[index - 1];
            float distance = delta.magnitude;
            float restLength = strandLength / (segmentsPerStrand - 1);
            if (distance > restLength)
            {
                Vector3 correction = delta.normalized * (distance - restLength) * 0.5f;
                positions[index] -= correction;
                positions[index - 1] += correction;
            }

            // Verlet
            Vector3 newPos = positions[index] + velocity * damping + force * deltaTime * deltaTime / masses[index];
            oldPositions[index] = positions[index];
            positions[index] = newPos;
        }
    }

    void Update()
    {
        var job = new HairPhysicsJob
        {
            positions = positions,
            oldPositions = oldPositions,
            masses = masses,
            deltaTime = Time.deltaTime,
            damping = damping,
            gravity = Physics.gravity
        };
        JobHandle handle = job.Schedule(positions.Length, 64);
        handle.Complete();
    }

    void OnDestroy()
    {
        positions.Dispose();
        oldPositions.Dispose();
        masses.Dispose();
    }
}

            upper_lip = [61, 185, 40, 39, 37, 0, 267, 269, 270, 409, 291]
            lower_lip = [146, 91, 181, 84, 17, 314, 405, 321, 375]
            lip_points = []
            for idx in upper_lip + lower_lip:
                x = int(landmarks.landmark[idx].x * w)
                y = int(landmarks.landmark[idx].y * h)
                lip_points.append([x, y])

            mask = np.zeros((h, w), dtype=np.uint8)
            cv2.fillPoly(mask, [np.array(lip_points)], 255)
            mask = cv2.GaussianBlur(mask, (21, 21), 0)

            foreground = np.zeros_like(image)
            foreground[mask == 255] = color
            result = cv2.addWeighted(image, 0.7, foreground, 0.3, 0)
            return result
        return image

    def process_frame(self, image):
        return self.apply_lipstick(image)
```

**Pros**: Real-time; accurate; open-source.  
**Cons**: 2D limits; lighting dependency; pose issues.  
**Integration**: Webcam batch; 3D export.
