### Assembly & Editing Agent Architecture

The Assembly & Editing Agent functions as a temporal reasoning system that analyzes narrative structure, emotional beats, and visual continuity to produce compelling sequences. It uses PyAV, a Python binding for FFmpeg, to access media through containers, streams, packets, codecs, and frames. This enables frame-accurate operations critical for professional post-production.

#### Core Video Processing Engine

PyAV provides performant FFmpeg library bindings, invoking libraries directly rather than executables, which optimizes processing of hundreds of clips. The agent leverages PyAV for handling input formats, codec conversions, and frame manipulations.

Key PyAV features include:
- **Container management**:
  ```python
  import av
  container = av.open('path_to_video')
  for frame in container.decode(video=0):
      frame.to_image().save(f'frame-{frame.index}.jpg')
  ```
- **Frame-accurate seeking**: Set `CodecContext.skip_frame` to process only keyframes for efficiency.
- **Threaded decoding**: Uses SLICE threading by default; multiple threads decode frames cooperatively for faster performance.
- **Color space conversion**: Creates new `VideoFrame` instances with specified width, height, format, and color space.

#### MLT Framework Integration

The Media Lovin' Toolkit (MLT) is an open-source multimedia framework for broadcasting, editing, and streaming. The agent integrates MLT via VidPy, a Python wrapper for video editing and compositing.

MLT architecture supports:
- **Multitrack editing**: Author and manipulate playlists, tracks, filters, and transitions via DOM.
- **Real-time processing**: Manages multitrack audio/video compositions.
- **Plugin system**: Extensible API with XML authoring and bindings for C++, Java, Lua, Perl, PHP, Python, Ruby, and Tcl.
- **Cross-platform support**: Runs on multiple operating systems.

VidPy enables composition:
```python
from vidpy import Clip, Composition
clip = Clip('input_shot.mp4')
clip.fadein(1)      # Fade in over 1 second
clip.fadeout(5)     # Fade out over 5 seconds
clip.glow()         # Add glow effect
clip.position(x=100, y=20)  # Set position
composition = Composition()
composition.add_clip(clip)
```

#### Pacing Analysis Algorithm

Editors control rhythm through shot duration, pauses, and transitions. The agent analyzes temporal rhythm, where average shot lengths in modern films range from 4-6 seconds as a neutral baseline. Deviations build intensity or provide relief.

**Shot Duration Analysis**:
- **Emotional intensity mapping**: Shortens shots to increase tension; lengthens for calm.
- **Genre conventions**: Fast pacing in action films like *Mad Max: Fury Road*; slower in dramas like *Lady Bird*.
- **Narrative progression**: Adjusts based on story arc.

**Rhythmic Pattern Recognition**:
Rhythm emerges from cut frequency and shot durations, complementing or contrasting scene content.

The agent evaluates:
- **Internal rhythm**: Emotional and narrative pace.
- **External rhythm**: Editing and cinematography-driven pace.
- **Temporal manipulation**: Expands or compresses time for suspense or emphasis.

#### Advanced Transition Selection

The agent selects transitions based on visual continuity, emotional impact, and narrative flow. Sound bridges maintain rhythm; rapid short shots accelerate action; varied lengths control pacing.

**Transition Decision Matrix**:
- **Cut selection**: Uses visual similarity, motion vectors, and emotional continuity.
- **Dissolve timing**: Synchronizes with music beats for cohesive audio-visual flow.
- **Creative transitions**: Applies wipes, morphs, or stylistic effects per genre and intent.

#### OpenTimelineIO Integration

OpenTimelineIO (OTIO) is an API and interchange format for editorial data, serving as a modern Edit Decision List (EDL). It references external media without storing it.

OTIO enables:
- **Cross-platform compatibility**: Adapters for Final Cut Pro XML, AAF, CMX 3600 EDL, and more.
- **Version control**: Tracks shot changes for collaboration.
- **Metadata preservation**: Retains timecodes and decisions.

Example:
```python
import opentimelineio as otio
timeline = otio.adapters.read_from_file('foo.aaf')
for clip in timeline.find_clips():
    print(clip.name, clip.duration())
```

### Color Grading Agent Architecture

The Color Grading Agent manages color intelligently, analyzing content, applying treatments, and ensuring compliance across formats. It builds on OpenColorIO (OCIO) for color management.

#### OpenColorIO Color Management

OCIO enforces color methodologies for high-fidelity imaging in production. It integrates with the Academy Color Encoding System (ACES), a device-independent standard.

Core OCIO features:
- **Color space transforms**: GPU renderer matches CPU for final frames.
- **LUT processing**: Supports Academy/ASC Common LUT Format (CLF); accurate for extreme exposures.
- **HDR support**: Properties for color space and transfer characteristics (including HDR) in consumers like DeckLink.

#### LUT Generation and Management

The agent generates custom LUTs using OCIO tools. LUTs transform colors, with 1D for simple adjustments and 3D for complex hue, saturation, and luminance changes. Shaper LUTs normalize inputs to 0-1 range.

**LUT Baking Process**:
Bake transforms for applications lacking native OCIO support, such as Autodesk Flame or Adobe Photoshop.

Example:
```bash
ociobakelut --format houdini --inputspace lg10 --outputspace srgb8 log_to_display.lut
```

#### Shot Matching and Consistency

The agent uses computer vision for consistency, extracting color statistics (histograms, RGB averages, dynamic range) and CLIP for visual similarity. It ensures temporal smoothness across cuts.

#### HDR and Wide Color Gamut Support

Supports modern workflows with:
- **HDR standards**: ITU-R BT.2100 tone mapping for HDR-to-SDR.
- **Color gamut**: Rec. 2020, P3, and Rec. 709.
- End-to-end 10-bit processing via compatible FFmpeg components.

#### Film Emulation Engine

Replicates film and video characteristics:
- **Film stock simulation**: Grain, color response (e.g., Kodak Vision3), contrast curves, and lens effects (aberration, vignetting).
- **Vintage video**: Analog artifacts (VHS tracking, color bleeding), broadcast standards (PAL/NTSC), and CRT effects.

### Pacing Analysis Algorithm
```python
import numpy as np
from scipy import signal
import cv2
from sklearn.ensemble import RandomForestRegressor

class PacingAnalyzer:
    def __init__(self):
        self.shot_duration_model = RandomForestRegressor(n_estimators=100)
        self.emotional_intensity_threshold = 7
        self.genre_pacing_profiles = {
            'action': {'avg_shot_length': 8, 'variance': 2},
            'drama': {'avg_shot_length': 4, 'variance': 1},
            'thriller': {'avg_shot_length': 6, 'variance': 8}
        }

    def analyze_shot_rhythm(self, timeline):
        """Analyze temporal rhythm patterns in edit sequence."""
        shot_durations = []
        motion_intensities = []

        for clip in timeline.find_clips():
            duration = float(clip.duration().value) / clip.duration().rate
            shot_durations.append(duration)
            motion_intensity = self.calculate_motion_intensity(clip)
            motion_intensities.append(motion_intensity)

        rhythm_profile = {
            'avg_shot_length': np.mean(shot_durations),
            'rhythm_variance': np.var(shot_durations),
            'motion_sync': self.calculate_motion_sync(shot_durations, motion_intensities),
            'pacing_score': self.calculate_pacing_score(shot_durations)
        }
        return rhythm_profile

    def calculate_motion_intensity(self, clip):
        """Calculate motion intensity using optical flow."""
        frames = self.extract_frames(clip, sample_rate=5)
        motion_vectors = []
        for i in range(1, len(frames)):
            flow = cv2.calcOpticalFlowPyrLK(frames[i-1], frames[i], None, None)
            motion_magnitude = np.mean(np.linalg.norm(flow[0], axis=1))
            motion_vectors.append(motion_magnitude)
        return np.mean(motion_vectors)

    def suggest_optimal_cuts(self, scene_analysis):
        """AI-driven cut suggestions based on pacing analysis."""
        suggestions = []
        for scene in scene_analysis:
            if scene['pacing_score'] < self.emotional_intensity_threshold:
                cut_suggestions = self.generate_cut_suggestions(scene)
                suggestions.extend(cut_suggestions)
        return suggestions
```

### Color Matching System
```python
import OpenColorIO as OCIO
import numpy as np
from sklearn.cluster import KMeans
import cv2

class ColorMatchingSystem:
    def __init__(self, ocio_config_path):
        self.config = OCIO.Config.CreateFromFile(ocio_config_path)
        self.processor_cache = {}

    def analyze_shot_color_characteristics(self, frame):
        """Extract comprehensive color statistics from frame."""
        lab_frame = cv2.cvtColor(frame, cv2.COLOR_RGB2LAB)
        color_stats = {
            'mean_rgb': np.mean(frame, axis=(0,1)),
            'std_rgb': np.std(frame, axis=(0,1)),
            'histogram_rgb': [np.histogram(frame[:,:,i], bins=256) for i in range(3)],
            'dominant_colors': self.extract_dominant_colors(frame),
            'color_temperature': self.estimate_color_temperature(frame),
            'dynamic_range': np.ptp(cv2.cvtColor(frame, cv2.COLOR_RGB2GRAY))
        }
        return color_stats

    def generate_matching_lut(self, reference_shot, target_shots):
        """Generate custom LUT for shot matching."""
        reference_stats = self.analyze_shot_color_characteristics(reference_shot)
        processor = self.config.getProcessor('scene_linear', 'sRGB')
        lut_size = 32
        lut_data = np.zeros((lut_size, lut_size, lut_size, 3))
        for r in range(lut_size):
            for g in range(lut_size):
                for b in range(lut_size):
                    input_color = [r/(lut_size-1), g/(lut_size-1), b/(lut_size-1)]
                    output_color = processor.applyRGB(input_color)
                    lut_data[r, g, b] = output_color
        return lut_data

    def apply_hdr_tone_mapping(self, hdr_image, target_nits=100):
        """Apply perceptual tone mapping for HDR to SDR conversion."""
        aces_transform = self.config.getProcessor('ACES2065-1', 'sRGB')
        sdr_result = aces_transform.applyRGB(hdr_image)
        tone_mapped = self.perceptual_tone_map(sdr_result, target_nits)
        return tone_mapped

    def extract_dominant_colors(self, image, k=5):
        """Extract dominant colors using K-means clustering."""
        data = image.reshape((-1, 3))
        kmeans = KMeans(n_clusters=k, random_state=42)
        kmeans.fit(data)
        colors = kmeans.cluster_centers_
        labels = kmeans.labels_
        color_percentages = []
        for i in range(k):
            percentage = np.sum(labels == i) / len(labels)
            color_percentages.append((colors[i], percentage))
        return sorted(color_percentages, key=lambda x: x[1], reverse=True)
```

### Rough Assembly Auto-Editor
```python
import opentimelineio as otio
import av
from vidpy import Clip, Composition

class RoughAssemblyEditor:
    def __init__(self, screenplay_path, shot_database):
        self.screenplay = self.parse_screenplay(screenplay_path)
        self.shot_database = shot_database
        self.quality_threshold = 8

    def auto_assemble_sequence(self, scene_list):
        """Automatically assemble rough cut from shot database."""
        timeline = otio.schema.Timeline()
        video_track = otio.schema.Track(kind=otio.schema.TrackKind.Video)
        for scene in scene_list:
            scene_shots = self.select_best_shots(scene)
            scene_sequence = self.assemble_scene(scene_shots)
            for shot_clip in scene_sequence:
                video_track.append(shot_clip)
        timeline.tracks.append(video_track)
        return timeline

    def select_best_shots(self, scene):
        """Select highest quality shots for scene assembly."""
        candidate_shots = self.shot_database.get_shots_for_scene(scene.id)
        scored_shots = []
        for shot in candidate_shots:
            quality_score = self.calculate_shot_quality(shot)
            if quality_score > self.quality_threshold:
                scored_shots.append((shot, quality_score))
        scored_shots.sort(key=lambda x: x[1], reverse=True)
        selected_shots = [shot for shot, score in scored_shots[:3]]
        return selected_shots

    def calculate_shot_quality(self, shot):
        """Calculate technical and artistic quality score."""
        container = av.open(shot.file_path)
        quality_metrics = {
            'technical_quality': self.assess_technical_quality(container),
            'performance_rating': shot.performance_metadata.rating,
            'composition_score': self.analyze_composition(container),
            'audio_quality': self.assess_audio_quality(container)
        }
        total_score = (
            quality_metrics['technical_quality'] * 3 +
            quality_metrics['performance_rating'] * 4 +
            quality_metrics['composition_score'] * 2 +
            quality_metrics['audio_quality'] * 1
        )
        container.close()
        return total_score

    def assemble_scene(self, shots):
        """Assemble individual shots into coherent scene sequence."""
        composition = Composition()
        for i, shot in enumerate(shots):
            clip = Clip(shot.file_path)
            if i == 0:  # Establishing shot
                clip = self.apply_establishing_shot_treatment(clip)
            elif i == len(shots) - 1:  # Final shot
                clip = self.apply_closing_shot_treatment(clip)
            else:  # Coverage shots
                clip = self.apply_coverage_treatment(clip, shot.shot_type)
            composition.add_clip(clip)
        return composition.render_timeline()
```

            if i > 0:
                clip.fadein(5)
            if i < len(clips_list) - 1:
                clip.fadeout(5)
            clip.position(offset=i * clip.duration)
            composition.clips.append(clip)

class AdvancedColorGradingAgent:
    def __init__(self, ocio_config_path):
        self.config = OCIO.Config.CreateFromFile(ocio_config_path)
        self.processor_cache = {}
        self.pyav_engine = PyAVColorProcessor()

    def process_shot_matching(self, shot_sequence):
        """Intelligent color matching using OCIO + PyAV."""
        reference_shot = shot_sequence[0]
        ref_container = av.open(reference_shot.path)
        ref_frame = next(ref_container.decode(video=0))
        ref_stats = self.extract_color_statistics(ref_frame)
        ref_colorspace = self.detect_colorspace(ref_stats)
        matched_shots = []
        for shot in shot_sequence[1:]:
            shot_container = av.open(shot.path)
            shot_frame = next(shot_container.decode(video=0))
            shot_stats = self.extract_color_statistics(shot_frame)
            transform = self.generate_matching_transform(shot_stats, ref_stats, ref_colorspace)
            corrected_shot = self.apply_color_transform(shot, transform)
            matched_shots.append(corrected_shot)
        return matched_shots

    def generate_hdr_deliverables(self, master_timeline):
        """HDR/SDR delivery pipeline."""
        hdr_processor = self.config.getProcessor('scene_linear', 'rec2100-pq')
        sdr_processor = self.config.getProcessor('scene_linear', 'rec709-gamma')
        dcp_processor = self.config.getProcessor('scene_linear', 'p3-dci')
        deliverables = {}
        for shot in master_timeline.find_clips():
            container = av.open(shot.file_path)
            hdr_frames, sdr_frames, dcp_frames = [], [], []
            for frame in container.decode(video=0):
                frame_array = frame.to_ndarray()
                hdr_frame = hdr_processor.applyRGB(frame_array)
                sdr_frame = sdr_processor.applyRGB(frame_array)
                dcp_frame = dcp_processor.applyRGB(frame_array)
                hdr_frames.append(hdr_frame)
                sdr_frames.append(sdr_frame)
                dcp_frames.append(dcp_frame)
            deliverables[shot.name] = {'hdr_master': hdr_frames, 'sdr_version': sdr_frames, 'dcp_version': dcp_frames}
        return deliverables

class UnifiedFrameFormat:
    def __init__(self, source_frame, source_format):
        self.source = source_frame
        self.format = source_format
        self._numpy_cache = None

    def to_pyav(self):
        """Convert to PyAV VideoFrame."""
        if self.format == 'pyav':
            return self.source
        return av.VideoFrame.from_ndarray(self.to_numpy())

    def to_numpy(self):
        """Convert to NumPy array for OpenColorIO."""
        if self._numpy_cache is None:
            if self.format == 'pyav':
                self._numpy_cache = self.source.to_ndarray()
            elif self.format == 'pil':
                self._numpy_cache = np.array(self.source)
        return self._numpy_cache

    def apply_ocio_transform(self, processor):
        """Apply OpenColorIO transform."""
        numpy_frame = self.to_numpy()
        transformed = processor.applyRGB(numpy_frame)
        return UnifiedFrameFormat(transformed, 'numpy')
